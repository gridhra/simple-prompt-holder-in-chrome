# プライバシーポリシー / Privacy Policy — Simple Prompt Holder

最終更新日: 2026-07-04

## 概要

Simple Prompt Holder（以下「本拡張機能」）は、ユーザーが頻繁に使うプロンプト文を
保存・検索し、クリップボードへコピーするための Chrome 拡張機能です。本拡張機能は
ユーザーのプライバシーを最優先します。

## 収集・送信する情報

**本拡張機能は、いかなる個人情報・利用データも収集、送信、共有しません。**

- 外部サーバーとの通信を一切行いません。
- アナリティクス、トラッキング、広告 SDK を一切含みません。
- 閲覧中のウェブページの内容を読み取ったり送信したりしません。

## データの保存

ユーザーが登録したプロンプト（タイトル・本文・タグ）は、`chrome.storage.local` を
用いて**ユーザーの端末内にのみ**保存されます。このデータが端末外に送信されることは
ありません。エクスポート機能で出力される JSON ファイルは、ユーザー自身の操作により
ユーザーのローカル環境に保存されるものです。

## 権限の利用目的

- **storage**: プロンプトを端末内にローカル保存するために使用します。
- **clipboardWrite**: 選択したプロンプト本文をクリップボードへコピーするために使用します。
- **ホストアクセス（全 URL）**: 任意のウェブページ上でオーバーレイ UI を表示する
  ためにのみ使用します。ページ内容の読み取り・収集・送信は行いません。

## データの削除

保存されたプロンプトは、拡張機能の UI から個別に削除できます。拡張機能を
アンインストールすると、`chrome.storage.local` 内のデータもすべて削除されます。
